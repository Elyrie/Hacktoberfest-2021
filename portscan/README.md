# Port Scan

This is a simple port scan writing in Python.

## Installation

```bash
pyenv virtualenv 3.9.4 portscan
pyenv local portscan
pip install -r requirements.txt
```

## Usage

```bash
python app/main.py <ip_address>
```

## Example

